# KSU Hostel Finder

**KSU Hostel Finder** is a full-stack web application for discovering and managing hostel accommodations near KSU.
It provides a modern, user-friendly interface for students to browse hostels and for admins to manage listings.

---

## What It Does

- **Browse Hostels:** Displays all available hostels in a clean, responsive grid layout.
- **Detailed Information:** Clicking on a hostel card opens a modal with comprehensive details, including pricing, advance, distance from campus, amenities, nearby facilities, address, contact information, and a Google Maps link.
- **Modern UI:** Uses Bootstrap 4 and a dark theme for an attractive and accessible user experience.

---

## Features

- **Student-Friendly Browsing:** Quickly find hostels with all essential details.
- **Rich Detail View:** See pricing, advance, amenities, facilities, contact info, and maps in a single modal.
- **Admin Control Panel:** Manage hostel listings easily through a graphical desktop interface.
- **Attractive Design:** Clean, responsive, and easy-to-use interface with a modern dark theme.
